# Spam Document Classifier

A Machine Learning project that classifies messages as Spam or Ham using Python and Scikit-learn.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Multinomial Naive Bayes

## Features

- Reads and preprocesses CSV dataset
- Converts text into numerical vectors
- Trains ML model using Naive Bayes
- Predicts whether a message is spam or not
- Accepts user input
- Shows model accuracy

## Workflow

1. Data Loading
2. Data Cleaning
3. Label Encoding
4. Train-Test Split
5. Text Vectorization using CountVectorizer
6. Model Training
7. Prediction
8. Accuracy Evaluation

## Run the Project

pip install pandas scikit-learn

python3 src/main.py
